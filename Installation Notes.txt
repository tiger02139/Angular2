1. Install Node JS
Go to https://nodejs.org/en/ and download and install version 8.9.1 LTS
2. Open cmd window and type "node -v" then "npm -v".  Verify you have at least Node 6.9 or higher 
and npm version 3 or higher.
3. Follow instructions at https://github.com/angular/angular-cli.  Type "npm install -g @angular/cli"
4. In the first lab at Labs\lab_Hello_Angular, you may need to run "npm install"
5. Type "ng serve"
6. Go to http://localhost:4200
